== Sample Kubeconfig 

Sample Kubeconfig to automatically scale reserved size and set the pod count, in this example 1000.

.yaml
[source,yaml]
----
apiVersion: machineconfiguration.openshift.io/v1
kind: KubeletConfig
metadata:
  name: dynamic-node
spec:
  autoSizingReserved: true
  kubeletConfig:
    maxPods: 1000
  machineConfigPoolSelector:
    matchLabels:
      pools.operator.machineconfiguration.openshift.io/master: ''
----
